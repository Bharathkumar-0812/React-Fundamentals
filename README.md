## Topics (high-level)

Fundamentals:
- JSX and the Virtual DOM
- Components (function vs class)
- Props and state
- Event handling and forms
- Composition and children

Core concepts / Important patterns:
- Hooks (`useState`, `useEffect`, `useMemo`, `useCallback`, `useRef`, custom hooks)
- Context API and patterns for global state
- Component composition and render props
- Controlled vs uncontrolled components
- Error boundaries and boundary patterns

Advanced topics:
- Concurrent React, Suspense, and streaming
- Server-side rendering (SSR) and hydration
- React internals: reconciliation and fiber
- Performance tuning and profiling
- Build-time optimizations (code-splitting, lazy, memoization)
- Testing strategies (unit, integration, e2e)
